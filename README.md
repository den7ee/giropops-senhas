Criar imagem (Dockerfile):

- WORKDIR /app
- COPY requirements.txt .
- COPY app.py .
- COPY templates templates/
- COPY static static/
- RUN pip install --no-cache-dir -r requirements.txt
- EXPOSE 5000
- CMD ["flask", "run", "--host=0.0.0.0"]

Buildar imagem:
- docker image build -t giropops-senhas:1.0 .
